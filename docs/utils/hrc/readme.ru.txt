HRC это файлы определяющие подсветку синтаксиса для программы colorer,
которая также реализована в виде плагина к FAR manager (и к многим другим
редакторам также). В этом каталоге лежат файлы hrc для форта для двух версий
colorer'а :
 forth.4patch2.hrc - для colorer 4ever patch2
 forth.take5.hrc - для colorer take5
Они улучшены по сравнению с дефолтными в поставке colorer'а.
Переименуйте нужный вам файл в forth.hrc и поместите в каталог colorer/hrc/base
Может понадобиться добавить больше расширений для автоматического включения подсветки
Форта colorer'ом - для этого открываем colorer/hrc/proto.hrc и в блоке
<prototype name="forth" ...
добавляем нужные расширения в строку
<filename>/\.(fth|f32|f|spf)$/i</filename>
