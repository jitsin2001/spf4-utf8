1. Как наследовать один класс от другого?

   Большинство ваших классов следует наследовать от класса Object - основного
   класса в системе. Делается это так:
   
   CLASS: MyClass <SUPER Object
     ...
     
   ;CLASS

   Хотя можно и не наследовать ни от кого, но в этом случае у вас не будет
   стандартных методов :new ...
    
2. Как мне наследовать метод? То есть я в новом классе переопределил
   метод и хочу из него вызвать старый (например :new)
   
   В большинстве языков программирования используется слово INHERIT в
   различных вариациях. Но удобно оказалось использовать слово own,
   которое к тому же используется еще для вызова методов данного класса.
   
   Например:
   
   CLASS: Test <SUPER Object
   
   : :new \ перегрузили :new
       own :new  \ вызвали старый
       ...
   ;    

3. Можно ли вызывать методы своего класса?

   Конечно. Причем есть 2 варианта:
   Первый - например нужно вызвать метод :method своего класса.
   Пишем - self :method
   Но - это вызов динамический, то есть метод ищется по ходу выполнения
   программы, а это медленно. Для этого придуман следующий вариант:
   Пишем - own :method
   Метод компилируется статически, то есть как обычное форт слово.
   Слово own оказалось удобным и для наследования методов.
   Но - помните, что own вызов хорош _только_ тогда, когда
   мы точно знаем (физически), какой именно метод должен вызываться.
