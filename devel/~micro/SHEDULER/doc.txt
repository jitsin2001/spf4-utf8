Данная программа предназначена для выполнения определённых действий
в некоторое время или через заданные интервалы времени.

Программа управляется файлом конфигурации sheduler.cfg, находящимся в
одном каталоге с программой.

Синтаксис:
\ после слэша идёт комментарий до коца строки
do
  <действия>
  \ эти действия будут выполняться в указанное время
every day|hour|min [ at [<часы> hour] [<минуты> min]]
\ или
do
  <действия>
  \ эти действия будут выполняться через указанный промежуток времени
thru <число> [ms|sec|min] \ при отсутствии единицы измерения -
\ миллисекунды

После блока do возможно несколько идущих команд every/thru, каждая на 
отдельной строке.

Действия.

В данной программе скомпилировано два отдельных проекта autopush для
нажатия Windows-ных button-ов и deleter для очистки каталогов. У
каждого из ниж свой набор команд.

Autopush

<hwnd> push - нажать на кнопку
  Для нахождения хандлера кнопки можно воспользоваться командами:
desktop - выдаёт хандлер десктопа
<hwnd> ->" <строка>" - найти хэндлер окна (или кнопки) дочернего от <hwnd>
и с заголовком <строка>
Таким образом команда выглядит следующим образом:
desktop ->" Заголовок окна" ->" Дочернее окно (если есть)" ->" кнопка" push

Deleter

del <каталог со слэшем>
\ Удалить все файлы из каталога

arch <каталог со слэшем> <файл>
\ Сохранять все файлы из каталога в файле
