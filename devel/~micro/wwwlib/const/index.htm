const.f

Описание кучи констант без лишних слов: 
CONST комментарий
  константа1 выражение1
  константа2 выражение2
  ..............................
  константаn выражениеn
;
Пустых строк быть не должно. Конструкция должна закончиться до конца файла.

vk_.f

Виртуальные коды (virtual codes) клавиш. Требуется: ~micro/lib/const/const.f.
