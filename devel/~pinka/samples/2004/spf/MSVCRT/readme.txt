 Буферизация на форте (FStream) быстрей работает, чем функции из MSVCRT.DLL

\ 24.Mar.2004

Относительная зависимость средней скорости от размера файлов такова:

 Для средних файлов (от ~7Кб)
   FStream работает в 7.6 раз быстрей, чем через WinAPI
   и в 1.5 раз быстрей, чем через MSVCRT.
 Для больших файлов (~80Кб): в 7.8 и 1.6 -соответственно.

 Таким образом, если FStream подключает файл размером size за 1 у.е.,
 то:
 size  WinAPI   MSVCRT
 ~800б  2.6       1.3
  ~7Кб  7.6       1.5
 ~80Кб  7.8       1.6
