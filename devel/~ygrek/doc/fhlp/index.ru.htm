Это справка SPF в CHM формате. Генерируется автоматически из файлов докуменатции в формате fhlp. Скрипты для сборки Вы можете найти на моей страничке. Сначала из fhlp файлов генерируются отдельные HTML для каждой секции, потом все эти HTML собираются в один CHM с помощью утилиты hhc.exe, входящей в HTML Help Workshop

Можно использовать просто как отдельный файл помощи. А для тех, кто использует FAR manager, есть ещё одна возможность - установить плагин cdhelps (Context dependent help) и настроить его на эту CHM'ку. Тогда прямо во время редактирования можно нажимать Shift-F1 на текущем слове (где стоит курсор) и получать контекстную справку - очень удобно!
